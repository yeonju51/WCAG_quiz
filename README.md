# 🌐 웹 접근성 실무 역량 평가 퀴즈 (KWCAG 2.2)

[한국형 웹 콘텐츠 접근성 지침 2.2(KWCAG 2.2)](https://nia-a11y.github.io/kwcag22tech/)을 기반으로 제작된 인터랙티브 실무 역량 평가 퀴즈입니다. 

웹 개발자, 퍼블리셔, 기획자, 디자이너가 실무에서 마주할 수 있는 접근성 이슈를 4가지 핵심 원칙(인식의 용이성, 운용의 용이성, 이해의 용이성, 견고성)에 따라 학습하고 점검할 수 있도록 설계되었습니다.

## ✨ 주요 기능 (Features)

* **실무 맞춤형 40문항**: KWCAG 2.2의 4대 원칙을 골고루 다루는 입문/중급/심화 난이도의 문항.
* **즉각적인 피드백 및 상세 해설**: 문제를 풀자마자 정답/오답 여부를 확인하고, 명확한 정답 근거와 실무 팁(💡)을 제공합니다.
* **📊 상세 결과 리포트 및 전체 문제지**: 퀴즈 완료 시(`/#result` 진입 시) 원칙별 달성률 등 상세한 점수 현황을 한눈에 파악할 수 있으며, 정답과 해설 숨기기/보이기가 가능한 **'전체 문제지'**를 통해 꼼꼼한 복습이 가능합니다.
* **오답 노트 & 미니 퀴즈**: 틀린 문제만 모아서 다시 푸는 '전체 오답 복습'과, 특정 원칙이 취약할 때 푸는 '핵심 복습' 기능을 지원합니다.
* **학습 편의성**:
  * 라이트/다크 모드 지원
  * 자동 넘김(Auto-advance) 켜기/끄기 설정
  * 로컬 스토리지(Local Storage)를 활용한 리더보드(결과 현황) 기록
* **접근성이 보장된 UI**: 퀴즈 인터페이스 자체도 키보드 탐색, 화면 낭독기(Screen Reader) 호환성, ARIA 속성, 포커스 관리 등 웹 접근성 지침을 준수하여 개발되었습니다.

## 🛠 기술 스택 (Tech Stack)

프레임워크나 외부 라이브러리(Dependencies) 없이 순수 웹 기술만로 가볍고 빠르게 동작하도록 개발되었습니다.

* **HTML5** (시맨틱 마크업 및 WAI-ARIA 적용)
* **CSS3** (반응형 디자인, CSS Variables, 애니메이션 처리)
* **Vanilla JavaScript** (상태 관리, DOM 조작, 키보드 이벤트 핸들링)
